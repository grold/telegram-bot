# RBAC Auth System Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement a database-backed Role-Based Access Control (RBAC) system with command-driven management.

**Architecture:** 
- **Database:** Expand `users` table with `role` and `is_authorized`. Add `command_permissions` table for dynamic access.
- **Middleware:** Replace `AdminMiddleware` with `AuthMiddleware` that checks the database for permissions.
- **Commands:** Add admin-only commands for managing users and permissions.
- **Migration:** Automated import from existing `.auth` file.

**Tech Stack:** `aiogram 3`, `sqlite3`.
